    • Cloud 소개
    Cloud는 기능별로 디스코드봇이 있다보니 복잡해집니다.
    하지만 "봇 1개에 최대한의 기능을 밀어넣으면 어떻까?" 라는생각에서 시작되어
    현재 게임, 도박, 서버관리, 대화, 밈 등 다양한 요소들이 있습니다.
    Cloud는 목표를 달성할때까지 업데이트를 멈추지 않을 예정입니다.
    Cloud는 멈추지 않습니다!!

    Cloud 사용법
    Cloud의 접두사는 총 3가지로 나뉩니다.
    일반 명령어 접두사 - //
    게임 명령어 접두사 - /.
    대화 접두사 - 구름아

    Cloud 문의, 신고, 질문
    Cloud 의 문의방법에는 2가지가 있습니다.
    1. 공식 디스코드 서버에 들어가 문의,신고,질문 하는방법
    2.//문의 를 이용해 봇으로 문의, 신고, 질문 하는 방법
    빠른 답변을 원하신다면 1번방법, 확실한 처리를 원하신다면 2번방법을 추천드립니다!
